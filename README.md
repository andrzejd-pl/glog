# Glog

Mini blog written in golang.

## How to run

```shell script
wget https://raw.githubusercontent.com/andrzejd-pl/glog/master/docker-compose.yml && \
docker-compose up -d
```

## Endpoints

- `GET /api/posts` - return all posts from database
